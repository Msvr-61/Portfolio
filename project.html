Twitter Spam Detection

Tools Used:Python, NLP, Scikit-learn, Pandas
Description:
This project focuses on detecting spam in Twitter posts using Natural Language Processing and Machine Learning techniques. The dataset was cleaned, vectorized using TF-IDF, and classified using models such as Naive Bayes and Logistic Regression. The study was documented and published in a research paper.Click on the image below to read the published paper.
Key Highlights:

  • Applied NLP techniques to preprocess and vectorize tweet data.
  • Trained ML models for binary classification (spam vs. not spam).
  • Evaluated using accuracy, precision, recall, and F1-score.
